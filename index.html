Daftar - Gratis Login

PeEeerr! adalah aplikasi web untuk membantu guru, siswa, dan orang tua siswa mengelola pengumpulan tugas belajar di rumah. Alat bantu ini merupakan bentuk sumbangsih kami untuk seluruh guru, siswa, dan orang tua siswa yang tengah menghadapi kesulitan akibat wabah COVID-19.

Alur Kerja

  • Guru memasukkan data tugas PR, untuk mendapatkan link upload.
  • Bagikan link dari PR tadi ke setiap siswa.
  • Siswa membuka link yang disediakan untuk memfoto hasil pengerjaan PR kemudian meng-upload foto-foto tersebut ke server.
  • Guru bisa melihat daftar tugas PR yang sudah di-upload dalam format file PDF (1 file per siswa).
